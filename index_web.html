Sobre mí

Desarrollo web con lectura administrativa y operativa.

Esta versión presenta a Hernán desde un perfil más elevado de soluciones digitales: diseño de páginas web, desarrollo de sistemas simples, automatizaciones y herramientas para gestionar información.

El diferencial está en unir criterio de gestión con ejecución técnica: no se diseña una pantalla aislada, se construye una herramienta que debe ser clara, útil y mantenible.
